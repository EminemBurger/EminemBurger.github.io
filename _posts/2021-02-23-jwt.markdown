## **JWT란**

JWT는 JSON Web Token의 약자로, JSON 데이터 구조를 암호화하는 방법으로 RFC7516 표준이다.  
주로 인증을 위해 사용되고, 보안을 위해 데이터 변환을 하는데도 사용된다.
